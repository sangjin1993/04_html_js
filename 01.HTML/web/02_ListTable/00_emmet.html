Emmet(ex zen-coding) 사용법

emmet 의 수식 형태

태그이름>포함될태그이름>...

    * : 동일한 태그를 뒤에 나오는 숫자 만큼 반복

    + : 연산자로 묶인 태그들을 서로 포함하지 않는 형태

    [속성명="값"] : 시작태그에 속성을 부여할 때 태그이름 뒤에 붙여서 작성

    () : emmet 식의 우선순위를 높이거나 묶어서 실행해야 하는 식을 결정

    {} : 태그 안쪽 컨텐츠 작성시 {} 안에 작성

    $ ~ $$ : 자동으로 증가하는 숫자 작성

emmet 식을 작성한 뒤 단축키 ctrl + e 로 실행
